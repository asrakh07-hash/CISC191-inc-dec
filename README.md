# Increment-Decrement JavaFX App
A simple JavaFX program that allows the user to input a number and increase it or decrease it by pressing one of the respective buttons. In this project, basic GUI interaction, input validation,
and JavaFX controller logic are demonstrated.
## Functionality
- Text input field for entering a number
- Increment and decrement buttons
- Handling of invalid input (defaults to O otherwise)
- Standard Maven project structure
## Usage/Running Program
### Required 
  - Java 17
  - Maven installed
### To Run 
From project root directory: 
`mvn javafx:run`
